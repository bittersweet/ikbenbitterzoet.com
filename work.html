Afstudeerproject

Om af te studeren van mijn opleiding Interactieve Media heb ik een project uitgevoerd bij Fullmoon in Amsterdam. Ik heb hun kunnen helpen om een professionele manier van werken in te voeren waarbij ik de resultaten ook nog in een adviesrapport heb gezet.

Tijdens mijn afstudeerstage heb ik samen met een andere stagiair de website van het bedrijf onderhanden mogen nemen. Zo heb ik de koppeling met het CMS verzorgt, nieuwe functionaliteit toegevoegd en vooral veel kunnen betekenen op het gebied van Javascript d.m.v. ieders favoriete library jQuery.
